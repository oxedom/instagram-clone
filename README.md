# instagram-clone

Instagram Clone by Sam.

# Features:

* Authentication 
* Custom Skeleton components
* Personal pet profiles
* fully responsive 
* Upload Images 
* Follow other pet!
* Get followed by other pets!
* Browse a feed of pet photos
* Get suggestions to follow pets
* Like a post
* Auto-complete searchbar
* comment on a post
